xvtouch Xvesa usb Touchscreen Linux driver

by Luc Hermans,   lh at dobit.com

     

Last update 15 May 2011
xvtouch Xvesa Xorg usb hid event udev Linux touchscreen driver

Introduction:
Simple Touchdriver Translates /dev/input/eventX into XtestFakeMotionEvent()
Suppose to work for all USB-HID absolute pointing devices
on Linux kernels supporting udev events
Calibration:
In xterm type:
  xvtouch /dev/input/eventX -c
Config:
Add to /home/tc/.xsession after waitforX
  killall xvtouch
  xvtouch /dev/input/event3 &
Add to /opt/.filetool.lst
  /usr/local/etc/xvtouch.cal
Requires:
Xorg-7.5-lib.tcz (Xlibs support libraries)
Copying-policy:
GNU GPL
Download:
Binary and source xvtouch.tgz
Development environment:
Tinycore Linux 3.5
install TC Apps: complitc, Xorg-7.5-lib, Xorg-7.5.dev [nano]
gcc -o xvtouch xvtouch.c -lX11 -lXtst -Wall
